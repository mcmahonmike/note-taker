# Note Taker

## Description
Note taker is an application that takes user input for a note title and note text to be displayed to a page and be saved to the page after the user leaves the page.

## Installation

Heroku was installed for running the server, as well as express.js, and the npm package uuid for the delete button function that is currently under production.

## Usage

The user provides a title and text for the note taker application and when they click the save icon the notes will be saved to the page.
